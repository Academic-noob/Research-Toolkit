# 📘 Pb–I 键长统计与八面体结构分析脚本使用说明

## 🧩 一、脚本功能简介

本脚本用于从 **VASP 计算结果**中提取 `.dat` 文件里的键长信息，并结合 `POSCAR` 结构：

- 自动识别 **Pb–I 配位关系（考虑周期性边界 PBC）**
- 区分 **轴向键（axial）** 和 **赤道键（equatorial）**
- 统计：
  - 平均键长
  - 最小键长
  - 最大键长
  - 出现次数
- 支持：
  - **单个 Pb 八面体**
  - **整体统计（Overall）**

---

## 📁 二、文件准备

请确保脚本运行目录下包含：
📂 当前目录
├── your_script.py # 本脚本
├── POSCAR # VASP结构文件（必须）
├── *.dat # 键长数据文件（可多个）

### 🔹 1. POSCAR 要求

- 必须是 **VASP 5+ 格式**
- 包含元素名（如 Pb I）
- 支持：
  - Direct 坐标 ✅
  - Cartesian 坐标（会自动转换）✅
- 支持 Selective dynamics ✅

---

### 🔹 2. `.dat` 文件格式要求

- 必须包含类似格式：
- distance = 3.21
- Pb1-I3 Pb1-I4

- distance = 2.95
- Pb2-I5

脚本会自动识别：

- `distance = xxx`
- 原子对：`Pb1-I3`

---

在脚本开头配置：
